# T.A.R.S: Raspberry Pi-Powered Replica from *Interstellar*


T.A.R.S is an ongoing project aimed at creating a functional replica of the robot from the movie *Interstellar*, utilizing a Raspberry Pi 4 as the central processing unit. The project is in its early stages, with foundational work underway.



## 🧠 Features


- 🔊 **Audio Output via Bluetooth Speaker** — ✅ *Completed*
- 🧾 **Conversation Logging** — ✅ *Completed* 
- 🎤 **Voice Input via Speech Recognition** — 🛠️ *In Progress*   
- 🗣️ **Text-to-Speech Response** — 🛠️ *In Progress*    
- 🧠 **Local LLM Integration (Alpaca, etc.)** — 🛠️ *In Progress*    
- 📟 **Terminal-based Interactive GUI** — 🛠️ *In Progress*    
- 🖱️ **OS Control (File Operations)** — 🛠️ *In Progress*    
- 🧪 **Servo Testing + Control** — 🛠️ *In Progress*    
- 📷 **Camera Module Support** — 🛠️ *In Progress*  
- 🤖 **Sensor Integration (Distance, etc.)** — 🛠️ *In Progress*  
- 💡 **Emotion/Intent Detection** — 🔜 *Not Started*  
- 🧩 **Plugin System for Extensibility** — 🔜 *Not Started*  
- 🌐 **Network Features / APIs** — 🔜 *Not Started*  
- 🗃️ **Personality Profiles + Memory** — 🔜 *Not Started*  
- 🧬 **Multi-Speaker Support** — 🔜 *Not Started*  
- 🖥️ **GUI-based Dashboard** — 🔜 *Not Started*



## 🧰 Hardware Used


- Raspberry Pi 4 (2GB RAM)
- MG996R Servo Motors  
- Ultrasonic Distance Sensors  
- 10000mAh Power Bank (5V 3A)  
- 5-Inch HDMI Display (800×480) with XPT2046 Touch Controller  
- Bluetooth Speaker  
- Raspberry Pi Official Camera (5MP)  
- Connecting Wires  
- Four 3.7V Li-ion Cells (7.4Wh total)  
- Six 18650 Li-ion Rechargeable Batteries (2600mAh)  
- Switches, Rheostats, Capacitors, etc.  
- 12V Cooling Fans  



## ⚡ Development Philosophy


- Lightweight & Local-first  
- Modular and Extendable  
- Raspberry Pi friendly  
- Inspired by real sci-fi HCI concepts
