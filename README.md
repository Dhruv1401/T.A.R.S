# T.A.R.S: Raspberry Pi-Powered Replica from *Interstellar*

**Project Overview**

T.A.R.S is an ongoing project aimed at creating a functional replica of the robot from the movie *Interstellar*, utilizing a Raspberry Pi 4 as the central processing unit. The project is in its early stages, with foundational work underway.

**Current Development:**

- Distance Sensor Integration(completed)
- Speech Recognition (ongoing)

**Upcoming Features :**

- Segmented Locomotion
- Omnidirectional Movement
- Speech Interaction
- Environmental Awareness
- Autonomous Operation
- Wireless Connectivity

**Hardware Components :**

- Raspberry Pi 4 2GB
- MG996R Servo Motor
- Ultrasonic Sensors
- 10000mAh Power Bank 5V 3A
- 5-Inch HDMI Display 800*480 Pixel XPT2046 Touch Controller
- Bluetooth Speaker
- Raspberry Pi Official Camera 5MP
- Connecting Wires
- Four 3.7V Li-ion Cell 7.4Wh
- Six 18650 Lithium Ion Rechargeable Batteries 2600mAh
- Switches , rheostats ,capacitors ,etc
- 12V Cooling Fans
- Bluetooth Speaker

**Software Overview :**

- Operating System
- Programming Languages
- Libraries and Frameworks
- RPi.GPIO / gpiozero
- OpenCV
- SpeechRecognition and pyttsx3
